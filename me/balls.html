2048: Ball Merge

Merge bouncy balls and get to 2048!

Drop balls into the playfield. Two balls of the same size merge!

2048: Ball Merge is a free physics-based puzzle game where you merge pairs of balls with the same point value to create one bigger, bouncier ball with the combined point value. The objective is to create a 2048-point ball.

The gameplay is based on Watermelon Game by popIn. The visual stylings are based on 2048.

Control schemes: mouse, touch

Languages: English, 日本語, español, Deutsch, français, русский
